# 🚗 Car Rental Web Application

A full-stack **Car Rental Web Application** that allows users to browse, view, and rent cars online with a smooth and responsive user experience.

🌐 **Live Demo:**  
👉 https://car-rental-jkkc.onrender.com

---

## 📌 About The Project

Car Rental is built using modern web technologies and follows best development practices.
It provides an easy-to-use interface for customers to explore available cars, make bookings online, and now get instant assistance through an integrated chatbot.

---

## 🚀 Features

### 🔎 Browse Available Cars
- View a catalog of cars
- See details like model, price, and availability

### 📱 Responsive Design
- Fully responsive UI
- Works on mobile, tablet, and desktop devices

### 📅 Booking System
- Users can reserve cars directly through the web interface

### 🛠 Admin Management
- Add new car listings
- Edit existing cars
- Delete cars (based on backend functionality)

### 🤖 AI Chatbot
- Real-time assistance for users
- Answers common queries about cars, pricing, and bookings
- Improves customer engagement and user experience
- Available directly on the website interface

## 📂 Project Setup (Run Locally)

Follow these steps to run the project on your local machine:

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/DevalVagadiya/Car_rental.git
cd Car_rental
```
### 2️⃣ Create & Activate Virtual Environment

### For macOS / Linux:

```bash
python -m venv venv
source venv/bin/activate
```
### For Windows:
```bash
python -m venv venv
venv\Scripts\activate
```
### 3️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 4️⃣ Apply Migrations
```bash
python manage.py migrate
```

### 5️⃣ Run Development Server
```bash
python manage.py runserver
```
---
## 📌 Notes
- Uses Django's default SQLite database (db.sqlite3)
- Requires Python 3.x
- Make sure pip is updated before installing dependencies
